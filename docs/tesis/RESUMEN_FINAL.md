## 🚀 Cómo Usar Todo Esto

### Opción 1: Probar el Plugin Funcional (5 minutos) ⭐

```bash
1. Abre: plugin-funcional.html
2. Crea tareas en la pestaña "Profesor"
3. Envía respuestas en "Estudiante"
4. Ve resultados en "Resultados"
5. Analiza plagio en "Plagio"
```

**Ventajas:**
- No necesitas instalar nada
- Funciona directamente en el navegador
- Evaluación real con OpenAI
- Perfecto para demos y presentaciones

---

### Opción 2: Ejecutar Pruebas Automáticas (1 minuto) ⭐

```bash
1. Abre: test-plugin-automatico.html
2. Click en "Ejecutar Todas las Pruebas"
3. Espera 30-60 segundos
4. Ve el resumen de resultados
```

**Ventajas:**
- Prueba todas las funcionalidades automáticamente
- Verifica que todo funciona correctamente
- Reporte visual de éxito/errores

---

### Opción 3: Instalar en Moodle (30-60 minutos)

```bash
1. Ejecuta: crear-zip-plugin.bat
2. Instala Moodle localmente (si no lo tienes)
3. Sube aiassignment.zip en Moodle
4. Configura tu API key
5. Prueba el plugin completo
```

**Ventajas:**
- Integración completa con Moodle
- Listo para producción
- Gestión de usuarios y cursos

## 🐛 Solución de Problemas

### Error: "Failed to fetch"
- Verifica conexión a internet
- Revisa que la API key sea válida
- Abre consola del navegador (F12)

### No se guardan los datos
- Verifica que localStorage esté habilitado
- No uses modo incógnito

### PHP no tiene extensiones
- Ejecuta: `habilitar-extensiones-php.bat` como admin
- O edita manualmente `php.ini`
