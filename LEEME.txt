🚀 INSTALACIÓN RÁPIDA DEL PLUGIN EN MOODLE (10 MINUTOS)

1. Crear ZIP del plugin:
   
   Windows:
   > scripts\crear-zip-plugin.bat
   
   Linux/Mac:
   $ ./scripts/crear-zip-plugin.sh

2. Instalar en Moodle:
   - Site administration → Plugins → Install plugins
   - Sube dist/aiassignment.zip
   - Actualiza base de datos
   - Configura tu OpenAI API Key

📖 Documentación: docs/instalacion/INSTALACION_RAPIDA.md

═══════════════════════════════════════════════════════════════

🧪 PROBAR SIN INSTALAR MOODLE (5 MINUTOS)

Si quieres probar el evaluador de IA sin instalar Moodle:

1. Configurar API Key:
   > cp .env.example .env
   (Edita .env con tu OPENAI_API_KEY)

2. Instalar dependencias:
   > npm install

3. Probar:
   > npm run demo

4. Abrir navegador:
   http://localhost:5000

📖 Documentación: demo-standalone/README.md

═══════════════════════════════════════════════════════════════

🌐 SERVIDOR WEB STANDALONE (10 MINUTOS)

Para usar el sistema sin Moodle:

1. Instalar dependencias:
   > npm install

2. Configurar:
   > cp .env.example .env
   (Edita .env con tu OPENAI_API_KEY)

3. Iniciar:
   > npm start

4. Acceder:
   http://localhost:5000

📖 Documentación: demo-standalone/README.md

🆘 AYUDA RÁPIDA

Problema: "No tengo API Key de OpenAI"
→ Obtén una en: https://platform.openai.com/api-keys

Problema: "No tengo Moodle instalado"
→ Opción 1: Usa test-environment/ para probar
→ Opción 2: Instala Bitnami Moodle Stack (30 min)
→ Opción 3: Usa el servidor standalone

Problema: "El plugin no se instala"
→ Revisa: moodle-plugin/INSTALACION_DESDE_INTERFAZ.md
→ Sección "Solución de Problemas"

Problema: "Quiero probar sin instalar nada"
→ cd test-environment
→ php demo-visual.php
